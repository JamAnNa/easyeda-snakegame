# easyeda-snakegame

## 项目简介

一个轻量级的贪吃蛇小游戏示例，适合作为前端/嵌入式演示或教学用例。包含游戏逻辑、简单的渲染与输入控制，易于移植与扩展。

## 主要特性

- 经典贪吃蛇玩法
- 键盘控制（方向键 / WASD）
- 支持速度与地图大小配置
- 简单易读的代码，便于二次开发

## 使用说明

- 开始游戏：页面加载后按下V按钮开始游戏
- 控制：方向键（↑↓←→）或 WASD。
